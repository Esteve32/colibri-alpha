Add a New Demo

To add a new demo to this gallery:

  1. Create a new folder in the demos/ directory
  2. Add your demo files (HTML, CSS, JS, etc.)
  3. Update the demos.json file with your demo information
  4. Commit and push your changes

The demo will automatically appear in this gallery!
